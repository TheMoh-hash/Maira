# 💄 Maira - Plateforme de Cosmétiques (En développement)

**Maira** est une interface e-commerce moderne dédiée à l'univers de la cosmétique. Ce projet a pour objectif de proposer une expérience utilisateur élégante et intuitive pour la sélection et l'achat de produits de beauté.

---

## 🏗 État du Projet
Actuellement, le projet est en phase de développement actif avec une attention particulière portée sur le **Frontend**. La structure est pensée pour s'intégrer dans une architecture **MERN** (MongoDB, Express, React, Node.js).

## ✨ Fonctionnalités Frontend
- **Interface Utilisateur Responsive :** Design optimisé pour une navigation fluide sur mobile et desktop.
- **Catalogue de produits :** Affichage dynamique des articles avec filtres de recherche.
- **Gestion du Panier (en cours) :** Mise en place de l'état global pour une gestion fluide des sélections.
- **Composants Réutilisables :** Architecture modulaire facilitant l'intégration future des appels API.

## 🛠 Stack Technique
- **Frontend :** React.js, Tailwind CSS (Stylisation), Axios (Prêt pour l'API).
- **Architecture :** Conception orientée composants pour une maintenance facilitée.
- **Prochaine étape :** Intégration du Backend (Node.js/Express) et persistance des données avec MongoDB.

## 🚀 Installation locale
1. Cloner le dépôt :
   `git clone https://github.com/TheMoh-hash/Maira.git`
2. Naviguer vers le dossier :
   `cd Maira`
3. Installer les dépendances :
   `npm install`
4. Lancer l'application :
   `npm start`
